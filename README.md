# alnah's Homebrew Tap

Private Homebrew tap repository prepared for future packages.

## Tap Name

Although the GitHub repository is named `homebrew-tap`, the Homebrew tap name is:

```bash
alnah/tap
```

## Install

```bash
brew tap alnah/tap
```

## Layout

- `Formula/` for command-line tools and packages installed with `brew install`
- `Casks/` for macOS applications installed with `brew install --cask`

## Status

No formulae or casks yet.

## Notes

Before adding the first package, make sure you have:

- a stable release artifact to download
- a versioning scheme
- a SHA256 checksum for each published artifact
